# Auto Order Project

## Overview
Automates order placement on a web app using Python and Selenium. Reads item details from Excel, places orders, and shows a bill popup with a Download PDF option.

## Features
- Automated login with visible typing
- Place orders from `items.xlsx`
- Browser alerts confirm orders
- Bill popup with download option
- Easy to extend

## Tech Stack
- Python 3.10+
- Selenium, Pandas, OpenPyXL
- Google Chrome + ChromeDriver
- Excel for input
