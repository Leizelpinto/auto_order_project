<h1><b>AUTO ORDER PROJECT</b><br></h1>
<h2><b> Overview </b><br></h2>
The Auto Order Project automates the process of placing orders on a web app using Python and Selenium. It reads item details from an Excel file (items.xlsx), places the order, and displays a bill popup with an option to download the bill in PDF format.

<h2><b>Features</b><br></h2>
<ul>
<li>Automated Login: Automatically logs into the web app with visible typing for better debugging and user experience.</li>
<li>Order Placement: Reads item details from items.xlsx and places orders accordingly.</li>
<li>Browser Alerts: Confirms order placement via browser alerts.</li>
<li>Bill Popup: After placing an order, a bill popup is displayed with an option to download the bill as a PDF.</li>
<li>Extensibility: The script is easy to extend, allowing for future updates and additional features.</li>
</ul>
<h2><b>Tech Stack</b><br></h2>
<ul>
<li>Python 3.10+<br>
<li>Selenium: For automating web interactions.</li>
<li>Pandas: For handling data manipulation.</li>
<li>OpenPyXL: For reading and writing Excel files.</li>
<li>Google Chrome + ChromeDriver: For web automation.</li>
<li>Excel: For input data (items.xlsx).</li>
</ul>
